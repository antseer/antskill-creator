# Color Pool — 可用主色池

> **每个 Skill 的 `--primary` 变量必须从本池选择**，且未被 `visual-registry.md` 占用。
> 本池之外的色值不可作为 Skill 主色；如果确实需要新色，请申请扩展本池（见 §4）。

## §2 配套次色

每个主色都有推荐的**次色**用于 chart 第二色系（主 > 次 > 第三 的顺序）。
次色由 Antseer 基础 palette 出（`#ffb000` / `#1196dd` / `#ffffff` / `#f44`），但 warning/danger 色**永远不变**（`#ffb000` / `#f44`），无论主色是什么。

```
chart_palette:
  - <主色>      (primary accent)
  - #ffb000     (amber — warning line / chart series 2)
  - #1196dd     (blue — info line / chart series 3)
  - #ffffff     (white — neutral line / chart series 4)
  - #f44        (red — danger line / chart series 5)
```

## §3 选色指南

### 按主题选

| 主题 | 推荐主色（按优先） |
|------|------------------|
| 风控 / 止损 / 警报 | 警报红 / 猩红 / 熔岩橙 |
| 研究 / 分析 / 学术 | 数据紫 / 琥珀 / 冰川蓝 |
| 监控 / 仪表盘 | 电气薄荷 / 信号黄 / 翡翠青 |
| 收益 / 投资组合 | 古铜金 / 生长绿 / 晨雾青 |
| 社区 / 治理 / 投票 | 海洋青 / 晨雾青 |
| 热度 / 新闻 / 趋势 | 熔岩橙 / 电光粉 |

### 禁用色

| 色 | 为什么禁用 |
|---|-----------|
| Antseer 电子绿 `#36dd0c` | 保留给 Antseer Hub 本体，Skill 不用 |
| 任何接近 `#05df72`（成功绿）的色 | 会与 candle up 混淆 |
| 任何接近 `#f44`（危险红）的色 | 会与 candle down / error 混淆 |
| 饱和度极高的纯蓝（`#0000ff` 级别）| 在暗底上刺眼，AA 对比失败 |

### 对比度校验

每个新入池的色值应在 `#080807` 底上做 WCAG 对比度校验：
- 对比度 ≥ 4.5:1 → 可用于大字 + CTA
- 对比度 ≥ 3:1 → 可用于线条 / chart / badge
- 对比度 < 3:1 → 不入池

## §4 扩展申请

当可用色剩 ≤ 2 个时，向工厂维护者申请扩展。
扩展时需提供：

1. 新色的 HEX 值
2. 色名（中文）
3. 气质 / 主题归属
4. `#080807` 底上的对比度数据
5. 与现有色池的视觉距离（不应与已有色过于接近）
