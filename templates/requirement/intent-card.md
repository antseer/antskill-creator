# Intent Card — 需求理解卡模板

> **配套方法论**: `methodology/intent-anchoring.md`
> **使用时机**: S0 进入 7 题画布**之前**,作为第一份产物给用户校验。
> **形态要求**: 既是 markdown 文档,也必须在 demo-v0.html 里以"屏 -1"渲染出来。

---

## 模板

```markdown
# 🧭 我对你需求的理解

## ① 你想解决的问题
> 一句话复述用户原话(最低限度改写,不要"翻译")

## ② 你的真实意图(推断)
- 不是 X(用户字面说的)
- 而是 Y(我推断的本质)
- 你最在意的可能是: A / B / C(列 1-3 个)

## ③ 关键决策点(2-4 个,带主张)

### 决策点 1: <问题>
- 选项 A
- 选项 B
- **我倾向 A,理由: <一句话>**

### 决策点 2: <问题>
- 选项 A
- 选项 B
- **我倾向 B,理由: <一句话>**

(最多 4 个决策点)

## ④ 我打算给你的
> 一句话定义最终交付物

---

[✓ 确认,继续生成]  [✎ 修改某项]  [✗ 完全重来]
```

---

## 写好理解卡的检查清单

| 检查项 | 通过标准 |
|---|---|
| ① 问题复述用户能认得出 | 70% 以上是用户原话 |
| ② 真实意图给出推断 | 不是把字面需求复读一遍 |
| ③ 决策点数量 | 2-4 个,不少不多 |
| ③ 每个决策点带主张 | 有"我倾向 X,理由 Y" |
| ③ 决策点是真模糊点 | 不是设计偏好(色/字体/布局这种留给 S2/S3) |
| ④ 一句话定义 | 能被用户复述出来 |
| 总长度 | 1 屏可读完,< 30 行 |

## 反例(不要这样写)

### ❌ 反例 1:开放式提问

```
我理解你想做财报分析。请问:
- 你想要看哪些股票?
- 你想要什么样的视觉?
- 你想要的数据精度是多少?
- 你想要的更新频率?
- 你打算每天用还是每周用?
```

为什么差:5 个开放式问题,用户没有任何决策路径,等于让用户自己写需求。

### ❌ 反例 2:无主张

```
关键决策点:
- 范围: 固定本周三场 / 自由组合
- AI 解读: 保守 / 中等 / 激进
- 数据: mock / 真跑
```

为什么差:没说倾向,用户得自己想清楚每个选什么。

### ❌ 反例 3:决策点是设计偏好

```
关键决策点:
- 主色用紫色还是蓝色?
- 字体 Inter 还是 Source Serif?
- 是否使用动效?
```

为什么差:这些是 S2/S3 工厂自动决策的,不该让用户管。

### ❌ 反例 4:把画布塞进卡片

```
我对你需求的理解:

WHO: 谁是目标用户?
WHAT: 核心功能是什么?
WHY: 为什么做?
WHERE: 数据从哪来?
...
```

为什么差:画布是工厂视角,用户视角的理解卡应该是"主张+决策点",不是空白填空题。

## 一个好例子(财报传导 skill)

```markdown
# 🧭 我对你需求的理解

## ① 你想解决的问题
> "本周三场美股财报对加密标的有什么影响?"

## ② 你的真实意图(推断)
- **不是**"看一份本周的财报周报"
- **而是**"建立一个能复用的传导分析能力,本周三场只是触发场景"
- 你最在意的可能是: 工具是否易拓展、是否诚实、是否能用图说话

## ③ 关键决策点

### 范围
- 本周三场固定 / 任意标的对自由输入
- **我倾向后者**,理由: 你前面说过"自由选择标的对,产品才不是一次性工具"

### AI 解读边界
- 只陈述事实 / 给概率指引 / 给操作建议
- **我倾向"只陈述事实"**,理由: 你说工具化、不替用户做判断

### 数据真假
- 全部 mock / 选 1-2 对真跑
- **我倾向 mock**,理由: S0 验证产品形态,不验证数据准确性。但 mock 要经得起加密分析师扫一眼

### 统计术语
- 显示 ρ / p / σ / 用大白话翻译
- **我倾向大白话**,理由: 用户听不懂的术语全部翻译或删除

## ④ 我打算给你的
> 一个"K 线叠加 + 财报锚点 + 历史回测 + 保守 AI 解读"的 stateless 工具,
> 本周三场是默认参数,不是边界。

---

[✓ 确认,继续生成]  [✎ 修改某项]  [✗ 完全重来]
```

这就是一张合格的理解卡:

- 复述了用户原话
- 推断了真实意图(不是字面复读)
- 4 个决策点都带主张和理由
- 一句话定义了最终交付物
- 不到 30 行,1 屏可读完
