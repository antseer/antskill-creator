方法论 · 漂移是怎么识别的?

漂移定义:概率突破 Bollinger 上/下轨(MA-20 ± 2σ)并维持 ≥ 24 小时。短于 24 小时的视为流动性噪音,自动过滤。

新闻节点来源:从 ant_news.by_market 按市场 ID 检索,按影响度过滤 Top 3。影响度规则:high = NYT/Reuters/Bloomberg 且互动 >1000;mid = 白名单来源或互动 >500。

入场判定:峰值后概率开始回落 + Z-Score 从 |2σ| 向 0 方向移动 → 标注「回归入场窗口」。若 Z-Score 持续扩大或反向突破,则标注「继续偏离」或「观望」。

风险提示
  • 本工具只做分析辅助,不构成投资建议。Polymarket 市场本身存在归零、延迟结算、流动性不足等风险。
  • 历史模式不保证未来重复:黑天鹅事件(重大司法/政治/战争)可能打破回归模式,此时 Z-Score 会持续扩大而非收敛。
  • 小盘市场漂移可能是操纵:若日成交量 < $100k,应警惕单笔大单制造假信号,建议忽略本 Skill 的判断。
  • 建议多源交叉:不要仅凭本工具下单,交叉参考真实新闻、盘面深度与对手盘质量。
